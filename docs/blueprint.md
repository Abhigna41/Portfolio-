# **App Name**: ProFolio

## Core Features:

- Project Showcase: Display a portfolio of projects with descriptions, images, and links.
- Skills Overview: Visually represent key skills and competencies.
- About Me Section: Personalized bio and photo to introduce yourself.
- Contact Form: Provide a contact form for visitors to send messages.

## Style Guidelines:

- Background color: Dark slate gray (#334756) for a modern, sophisticated look.
- Primary color: Muted Blue (#2C7DA0) evoking professionalism and reliability. 
- Accent color: Soft yellow (#FFE45E) to highlight key elements and calls to action.
- Body and headline font: 'Inter' sans-serif for clean and accessible typography.
- Simple, outlined icons to represent skills and project categories.
- Grid-based layout for organized presentation of projects and information.
- Subtle transitions for loading content.
